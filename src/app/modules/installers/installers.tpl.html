Bloom currently requires Microsoft Windows, but doesn't require any money, nor an internet connection.

Prerequisites

  • Windows 7 SP1, 8, 8.1
  • We don't test XP SP3 or Vista SP1 anymore, but they worked last time we did.
  • .Net 4.0
  • If you're using a Mac, you can run Bloom using Parallels or another Virtual Machine
  • We're hard at work on a Linux version.
  • Adobe Reader (Acrobat). Has been tested with Versions 11 (XI), 10 and 7. Here's where to find standalone/offline installers. Sorry, your favorite alternative reader is not a substitute (the forthcoming Bloom 2.1 version will remove this requirement).
  • (Optional) Art Of Reading

Font

By default, Bloom uses SIL's Andika literacy font. Do you need a version with one or more characters shaped differently to fit how people write in your language? You can also easily create your own version by selecting among alternate character shapes. Of course, you can also change the font of a book collection to something else.
