Bloom 3 installation instructions

Installing Bloom 3 and Related Software

If you have been using Bloom version 1 or 2, or an earlier version of Bloom 3, please uninstall the software before you install version 3. Bloom requires Windows XP SP3, Vista SP1, 7 SP1, 8, or 8.1 and .NET 4.0 (part of Windows, so you probably already have it). See these instructions for installing on Linux.

The Art of Reading

We also recommend that you download and install the Art Of Reading illustration collection. This free collection contains more than 10,000 line drawings suitable for use in reading materials. Bloom is designed to enable you to search and select illustrations from this collection once you have installed it.

Fonts

Bloom installs SIL’s Andika font which was designed specifically for easier reading in roman scripts. You can also use a font you already have installed on your computer, but it needs to be a Unicode font. SIL offers free Unicode fonts for a variety of needs:

  • Andika, designed for literacy materials, especially for beginning readers
  • Annapurna, for languages that use Devanagari script
  • Scheherezade, an extended Arabic script font
  • Abyssinica, an Ethiopic script font (also sometimes called “fidel” script)

Bloom assumes that you already have a way to type your language. If you need a way to type your language and lack one, you might consider Keyman, available in a free edition.
